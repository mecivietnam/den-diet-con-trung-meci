# Website MECI - Đèn diệt côn trùng

## Đăng lên GitHub Pages
1. Tạo repository mới.
2. Upload toàn bộ nội dung trong thư mục này vào nhánh `main`.
3. Vào Settings → Pages.
4. Chọn Deploy from a branch, nhánh `main`, thư mục `/root`.
5. Save và chờ GitHub cấp URL.

File `index.html` nằm ở thư mục gốc và `.nojekyll` đã được thêm để GitHub Pages phục vụ file tĩnh trực tiếp.
